# **App Name**: RankForge AI

## Core Features:

- AI-Powered Content Generation: Generate SEO-optimized articles and outlines, leveraging an intelligent tool that incorporates E.E.A.T. and G.E.O. principles based on user-provided topics, keywords, and audience insights.
- Real-time SEO Analysis & Scoring: Integrate with an external service to provide real-time SEO scoring and critical optimization suggestions, similar to Rank Math, as content is being created or edited.
- Intuitive Content Editing Interface: A dedicated rich text editor designed for refining AI-generated drafts, seamlessly implementing optimization suggestions, and structuring articles efficiently for readability and SEO.
- Dynamic Optimization Suggestions Tool: Offers dynamic, actionable recommendations using an intelligent tool to improve content for E.E.A.T., G.E.O., readability, keyword density, and overall search engine performance.
- Article Export Functionality: Enable users to export finalized, optimized articles in common formats such as HTML or Markdown for easy integration into their existing Content Management Systems.

## Style Guidelines:

- Primary color: A vibrant, professional blue (#2289CC). This hue evokes trust, technology, and clarity, ideal for a data-driven tool.
- Background color: A subtly desaturated, near-white light blue (#F7F9FA). This provides a clean, open canvas that keeps focus on the content without feeling stark.
- Accent color: A fresh, energetic green (#29A349). This color is chosen to signify growth and positive results, contrasting distinctly yet harmoniously with the primary blue.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable design, ensuring clarity across all content and interface elements.
- Utilize sleek, modern, and outline-style icons that maintain the application's professional aesthetic and enhance intuitive navigation.
- Implement a clean, spacious layout with ample white space to improve readability and reduce cognitive load during content creation and editing.
- Incorporate subtle, functional animations for feedback and transitions, enhancing the user experience without being distracting.
